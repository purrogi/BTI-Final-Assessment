Rogan Kim Final Assesment Reflection

Responsive design is used when web pages load onto various platforms and are able to adapt. Different platforms that support different layouts will differ for each. It allows for the display to automatically adjust and fit the current screen. During my research, I discovered that in order to have a responsive design, the developer will make only one source code but modify the style sheets for each unique platform.

The design I used is modular which allows it to be used on various platforms. This relates back to the responsive design.

I found it difficult because there are so many different screen types. Meaning, the pixel count is different in a lot of different devices. There are monitors, hundreds of phones, hundreds of tablets, etc. The list goes on and each one of them can have different size of screens which makes it complicated.

I thouroghly enjoyed this course. Although it's not necessarily a coding language, learning how HTML works, was very beneficial to me as I now pay attention to how every single web page is made and how I could possibly do it myself. I did find javascript to be a little challenging, but it was okay once I got caught up and practiced more and more. Even going out of my way to learn through third party websites to gain a better understanding of the language.
